1. Top 3 Overall Students

Conditions:

  • Every subject mark > 60
  • No absent marks in any subject

MySQL Query:

-- Simple query without window functions SELECT student_id, student_name, science_marks, maths_marks, english_marks, (science_marks + maths_marks + english_marks) AS total_marks, ROUND((science_marks + maths_marks + english_marks)/3, 2) AS average_marks FROM students WHERE science_marks > 60 AND science_marks != 'A' AND maths_marks > 60 AND maths_marks != 'A' AND english_marks > 60 AND english_marks != 'A' ORDER BY total_marks DESC LIMIT 3;

2. Top 3 in Each Subject

Conditions:

  • Rank by subject marks
  • Use total marks as tiebreaker
  • Exclude absent marks

MySQL Queries:

-- Science top 3 (simple version) SELECT student_name, science_marks AS marks, (science_marks + maths_marks + english_marks) AS total_marks FROM students WHERE science_marks != 'A' ORDER BY science_marks DESC, total_marks DESC LIMIT 3; -- Maths top 3 (simple version) SELECT student_name, maths_marks AS marks, (science_marks + maths_marks + english_marks) AS total_marks FROM students WHERE maths_marks != 'A' ORDER BY maths_marks DESC, total_marks DESC LIMIT 3; -- English top 3 (simple version) SELECT student_name, english_marks AS marks, (science_marks + maths_marks + english_marks) AS total_marks FROM students WHERE english_marks != 'A' ORDER BY english_marks DESC, total_marks DESC LIMIT 3;
