Type Command Description
DDL CREATE Creates new tables, views, or databases.
DDL ALTER Modifies an existing database object (e.g., table, view).
DDL DROP Deletes tables, views, or databases.
DML INSERT Adds new data to a table.
DML UPDATE Modifies existing data in a table.
DML DELETE Deletes data from a table.
DCL GRANT Gives user access privileges to database objects.
DCL REVOKE Removes user access privileges.
TCL COMMIT Saves all changes made during the current transaction.
TCL ROLLBACK Undoes changes made during the current transaction.
TCL SAVEPOINT Sets a savepoint within a transaction to allow rolling back to that point.
TCL SET TRANSACTION Sets the characteristics of the current transaction.
